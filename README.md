INTRODUCTION
In today’s education system, online exams are very common. Students give exams using computers or mobile phones. In such exams, time management plays a very important role. Even if a student knows all answers, poor time management can reduce marks.

Many students spend too much time on one section and very little time on another. Because of this problem, I decided to work on a minor project that analyses how students use their time during online exams.

In this project, I used Python programming to create sample exam data, analyze it, and show the results using graphs. This project helps in understanding exam time usage in a simple and practical way.

MAIN OBJECTIVE OF THIS PROJECT
Objective of the Project
The main objectives of this project are:

To understand the importance of time management in online exams
To create sample exam time data for students
To analyze time spent in each exam section
To calculate total and average exam time
To visualize exam time using graphs

This project is designed for learning purpose and helps beginners understand real life data analysis.

TECHNOLOGIES USED IN THIS PROJECT
This project is completely done using Python. The following libraries are used:

Python
Python is a simple and easy programming language. It is widely used for data analysis and real-world applications.
NumPy
